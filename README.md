## 🧠 Опис функціоналу

Проєкт складається з трьох незалежних скриптів для різних наборів даних. Вони реалізують наступний функціонал:

- **Завантаження даних**: Автоматичне завантаження наборів даних (CIFAR-10, CIFAR-100, Fashion-MNIST) з офіційних джерел.
- **Попередня обробка**: Перетворення зображень у відтінки сірого, нормалізація, формування бінарних задач "один клас проти всіх".
- **Архітектури мереж**:
    - **MLP**: Для CIFAR-10 та CIFAR-100 (один прихований шар, сигмоїд).
    - **CNN**: Для Fashion-MNIST (згортковий шар + GAP + сигмоїд, реалізований через NumPy).
- **Оптимізація Adam**: Повна реалізація оптимізатора з нуля, яка повертає фактичні кроки оновлення ваг.
- **Аналіз динаміки**: Обчислення вектора `X`, де кожен елемент — це середньоквадратичне значення (RMS) величини кроку оновлення Адама для відповідного нейрона або фільтра.
- **Візуалізація**: Побудова кольорових карт у просторі `(β₂, β₁)`, де колір визначається складним правилом, заснованим на послідовності норм корекцій ваг (`D[i] = X[i+1]² + X[i]²`).
- **Збереження результатів**: Експорт згенерованих карт у формати PNG (зображення) та CSV (дані про кольори).
